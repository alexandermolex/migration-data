# migration-data
This is going to be the basis for my data anaylsis project for SITD Python V
I will list the required libraries below. The xlsx and xls files needed to run this program are included in the repository but can also be found on the United States Census website (census.gov)
Libraries: TKinter, Pandas, Numpy, and OS (for redundencies sake)
Please also install the xlrd library via pip install or zip file. The Pandas library needs this since files in the earlier years (specifically 05-09) need xlrd to convert the xls into dataframes by which we can clean/reorganize them for use in plotting migration patterns.
